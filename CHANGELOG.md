# Changelog

## Поиск

### Префиксный поиск
- Добавлен поиск по части слова: `свобо` находит «свобода», «свободы», «свободно»
- Реализовано через FTS5-оператор `*` — каждый токен запроса дополняется звёздочкой
- Результаты отсортированы по `article_number ASC` вместо релевантности FTS

### Подсветка совпадений
- FTS5 `snippet()` подсвечивает найденное слово целиком, даже если введена только часть
- Добавлена пост-обработка `trimHighlight()`: `<b>президент</b>` → `<b>президе</b>нт` при запросе «президе»

## Система редкости

Каждой статье присвоена редкость в стиле игровых предметов (статический объект `RARITIES` во фронтенде, без изменений БД):

| Редкость | Цвет | Кол-во |
|---|---|---|
| Легендарная | Оранжевый `#f57c00` | 6 |
| Эпическая | Фиолетовый `#9c27b0` | 15 |
| Редкая | Синий `#1e88e5` | 25 |
| Необычная | Зелёный `#43a047` | 40 |
| Обычная | Серый `#9e9e9e` | ~51 |

- Бабл с редкостью отображается справа в строке карточки поиска
- В открытой статье бабл стоит рядом с «Глава N» в шапке

## Кнопка «Мне повезёт»

- Кнопка на главном экране под поиском
- Запрашивает `GET /random` → `ORDER BY RANDOM() LIMIT 1` в SQLite
- Открывает случайную статью в модале

### Конфетти при редких результатах
- При выпадении эпической статьи: 60 фиолетовых конфетти летят снизу вверх
- При выпадении легендарной статьи: 90 оранжево-золотых конфетти, более мощный залп
- Реализовано на чистом Canvas без сторонних библиотек — частицы с гравитацией и вращением, самоудаляются после завершения анимации
- Для обычной/необычной/редкой редкости анимация не запускается

## Избранное

- Звёздочка (SVG) в правом верхнем углу открытой статьи
- Пустая по умолчанию, заполненная оранжевая — если добавлено
- Анимация при нажатии: `scale(1) → scale(1.5) → scale(1)` за 0.28s
- Данные хранятся в `localStorage` (`favs`) — сохраняются между сессиями
- Хранится: `id`, `article_number`, `chapter`, `title` — отображение без дополнительных запросов
- Кнопка «Избранное» на главном экране рядом с «Мне повезёт», показывает счётчик
- Экран избранного — bottom sheet со списком карточек; тап открывает статью
- Пустое состояние: подсказка «Нажмите ★ при чтении статьи»
